# Hoteles App

Proyecto FrontEnd para elmundo desarrollado en Angular version 7.3.8

## Pre-Requisitos

Node y npm: https://nodejs.org/es/ 
Angular Cli: https://cli.angular.io/

## Instalacion

Ejecuta `npm install`

## Servidor de desarrollo

Ejecuta `ng serve`. Navega a ` http://localhost:4200/`.
